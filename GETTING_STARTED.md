# Collie - Getting Started

Welcome to Collie! This guide will help you get started with the application.

## Opening a Workspace

1. Click **File > Open Folder** or press `Ctrl+O`
2. Select a folder containing your product documentation
3. The file tree will appear in the left sidebar

## Creating Documents

### From Templates
Use one of our built-in templates:
- **PRD Template** - Product Requirements Document
- **Tech Spec Template** - Technical Specification
- **User Story Template** - User Story format

Access templates via:
- Menu: **Templates > New [Template Type]**
- Command Palette: `Ctrl+Shift+P` then search for template

### New File
- Menu: **File > New File** or `Ctrl+N`
- Sidebar: Click the **+** button

## Editing Features

### Monaco Editor
Collie uses Monaco Editor (the same editor as VSCode) with:
- Syntax highlighting for multiple languages
- IntelliSense and auto-completion
- Find and replace (`Ctrl+F`)
- Multi-cursor editing
- Minimap navigation

### Auto-Save
Files are saved automatically when you press `Ctrl+S`

## File Operations

### In the File Tree
- **Click** - Open file
- **Right-click** - Context menu (Rename, Delete)
- **Expand/Collapse** - Click folder icon

### Creating Files/Folders
1. Click the **+** button in the sidebar header
2. Enter the name
3. Choose **File** or **Folder**

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+O` | Open Folder |
| `Ctrl+N` | New File |
| `Ctrl+S` | Save File |
| `Ctrl+Shift+P` | Command Palette |
| `Ctrl+F` | Find in File |

## Command Palette

Press `Ctrl+Shift+P` to open the command palette and quickly access:
- Template creation
- File operations
- And more!

## Tips

- Use `.prd` extension for Product Requirements Documents
- Markdown files (`.md`) get preview support
- The status bar shows file info at the bottom
- Right-click files for quick operations
